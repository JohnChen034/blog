# Welcome to The Log

![logo](/img/logo.png)

创建这个博客主要是想要记录我的一些想法, 并把当下的想法写下来以让后续的我可以借鉴.另一个原因是我很容易忘事, 容易记住感觉而不是具体的想法, 所以在我需要的时候可以有一个站点来回顾. 


灵感来自 [Ground Oddity](https://idle.systems/). 原本我自己做了一个小网站, 但是我想要search bar because I record my thoughts here in case i waste time thinking about it in the future, so a search bar can retrieve my thoughts. 我自己做这个功能, 太耗时间. 不过以后闲下来我可以更customize everything.
