# JohnChen-blog

创建这个博客主要是想要记录我的一些想法, 并把当下的想法写下来以让后续的我可以借鉴.另一个原因是我很容易忘事, 容易记住感觉而不是具体的想法, 所以在我需要的时候可以有一个站点来回顾. 灵感来自 [Ground Oddity](https://idle.systems/). 

基本上我会想到什么就说什么, 希望逻辑不会显得混乱. 

# Usage:

I made a few changes to automize the pipeline so I just need to enter `make`/`mkdocs` then everything resolves. 

- `make new`: create new formatted markdown
- run `make prep` under source dir to 
    - rename markdown based on name and date
    - update index.html
- `mkdocs serve --livereload`: create local server that reloads when save
